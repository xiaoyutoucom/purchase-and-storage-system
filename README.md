# Purchase & Storage System

**English** | [简体中文](./README.zh-CN.md)

Purchase and warehouse system that integrates multi-platform ordering/inquiry/orders, uses scheduled jobs to refresh data, and stores order data on AWS.

| Platform | URL |
|----------|-----|
| **GitHub** | https://github.com/xiaoyutoucom/purchase-and-storage-system |

## Features

- Multi-platform order / inquiry integration
- Scheduled data refresh
- AWS-backed order storage

## Tech stack

- Java · Maven

## Build & run

Open the Maven project, configure platform credentials, AWS, and datasource settings, then run the Spring/Java main module.
