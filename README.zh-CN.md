# 采购仓储系统

[English](./README.md) | **简体中文**

采购仓储系统：对接多平台下单、询价与订单；调度服务自动更新数据；订单数据可落到 AWS 存储。

| 平台 | 地址 |
|------|------|
| **GitHub** | https://github.com/xiaoyutoucom/purchase-and-storage-system |

## 功能

- 多平台下单 / 询价 / 订单对接
- 调度服务自动更新数据
- AWS 存储订单数据

## 技术栈

- Java · Maven

## 编译运行

打开 Maven 工程，配置各平台凭证、AWS 与数据源后，运行主 Java/Spring 模块。
